# A Psalm plugin for Pimple (ArrayAccess). Ex: Pimple\Container

This is an plugin for Psalm to resolve return type for Pimple Container (ArrayAccess)

## Installation

Install with:

```
composer require --dev fcpl/psalm-pimple-container-plugin
```
